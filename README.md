# Para utilizar este proyecto debes:
- Clonar el repositorio: "https://github.com/MarceloOrtegaDev/practicos_multiplicar_sumar_Matrices.git"
- Utilizar uno de los backends

  ## Python
  ```
  Cd bacckendPy
  python -m venv venv
  pip install
  python app.py
  ```

  ## Javascript
  ```
  cd backend
  npm install
  npm run dev
  ```

## Frontend
```
npm install
npm run dev

```
